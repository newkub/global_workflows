## Goal

ทำตาม rules ใน `.windsurf/rules` ให้ครบถ้วน

## Execute

### 1. Read Rules

อ่านและเข้าใจ rules ทั้งหมด

1. อ่านไฟล์ rules ทั้งหมดใน `.windsurf/rules`
2. เข้าใจความหมายและวัตถุประสงค์ของแต่ละ rule

### 2. Apply Rules

ปฏิบัติตาม rules ที่กำหนด

1. ทำตาม rules ทั้งหมดตามลำดับ
2. ลบ bullet เมื่อทำเสร็จแล้ว
3. ตรวจสอบว่า rules ครบถ้วน

## Rules

### 1. Rule Compliance

- ทำตาม rules อย่างเคร่งครัด
- ลบ bullet เมื่อทำเสร็จแล้ว
- ตรวจสอบว่า rules ครบถ้วน

### 2. Rule Priority

- ทำตาม rules ตามลำดับที่กำหนด
- ถ้า rule ขัดแย้งกันให้ถาม user

## Expected Outcome

- Rules ใน `.windsurf/rules` ถูกทำตามครบถ้วน
- Bullet points ถูกลบเมื่อทำเสร็จแล้ว
