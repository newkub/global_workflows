## Goal

ตอบในแชทเป็นตารางที่มี numbered columns, grouping, sorting เพื่อความชัดเจน

## Execute

### 1. Prepare Data

1. รวบรวมข้อมูลที่ต้องการรายงาน
2. จัดกลุ่มข้อมูลตาม category
3. กำหนดลำดับความสำคัญ

### 2. Create Table

1. สร้างตารางพร้อม numbered columns
2. ใช้ numbered list สำหรับ columns
3. เรียงลำดับข้อมูลตามความสำคัญ

### 3. Group and Sort

1. จัดกลุ่มข้อมูลตามหมวดหมู่
2. เรียงลำดับภายในกลุ่ม
3. ใช้ headers สำหรับ grouping

### 4. Summarize as Numbered Items

1. แปลงข้อมูลในตารางเป็น numbered list
2. ใช้ format `1.`, `2.`, `3.` สำหรับแต่ละข้อ
3. จัดกลุ่มข้อตาม category
4. เรียงลำดับข้อตามความสำคัญ

### 5. Validate Output

1. ตรวจสอบว่าทุก column มี number
2. ตรวจสอบ grouping และ sorting
3. อ่านให้เข้าใจง่าย

## Rules

### Table Format

รูปแบบตารางที่ต้องใช้

- ทุกตารางต้องมี numbered columns
- ใช้ grouping และ sorting เสมอ
- สามารถมีหลายตารางได้
- ใช้ headers ชัดเจนสำหรับแต่ละ column

### Output Channel

ช่องทางที่ใช้ส่งรายงาน

- ตอบกลับในแชทเท่านั้น
- ไม่สร้างไฟล์แยกสำหรับรายงาน
- ใช้ markdown table format
- ตารางต้องอ่านง่ายบนทุก device

### Numbered Items

การจัดรูปแบบ numbered list

- ใช้ format `1.`, `2.`, `3.` สำหรับแต่ละข้อ
- จัดกลุ่มข้อตาม category
- เรียงลำดับข้อตามความสำคัญ
- แต่ละข้อต้องกระชับ ตรงประเด็น

## Expected Outcome

- ตารางที่มี numbered columns
- Grouping และ sorting ที่ชัดเจน
- ข้อมูลที่อ่านง่าย
- สรุปเป็น numbered list ที่ชัดเจน
