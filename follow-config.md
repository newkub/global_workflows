## Goal

ดู dependencies และ config files แล้วตั้งค่าตามที่จำเป็น

## Execute

1. ดู dependencies และ config files ใน project
2. เช็คใน global workflows และ skills ว่ามีอะไรให้ทำตามบ้าง
3. รันเฉพาะที่จำเป็นตาม stack ที่ใช้

## Rules

- รันเฉพาะที่จำเป็น ไม่รันทุก workflow ให้เหมือนกันหมด
