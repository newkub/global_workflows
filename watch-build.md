## Goal

watch build แล้ว fix error ทั้งหมดจนไม่มีเหลือ

## Execute

### 1. Watch Build

watch build process

1. เริ่ม watch build
2. ตรวจสอบ output อย่างต่อเนื่อง
3. ระบุ errors เมื่อเกิดขึ้น

### 2. Fix Errors

แก้ไข errors ทั้งหมด

1. ระบุ errors ทั้งหมด
2. แก้ไข errors ทีละตัว
3. ทำซ้ำจนกว่าไม่มี errors เหลือ

## Rules

- watch build อย่างต่อเนื่อง
- แก้ไข errors ทั้งหมดจนไม่มีเหลือ
- ตรวจสอบว่าไม่มี errors ที่เหลือ

## Expected Outcome

- Build สำเร็จ
- ไม่มี errors เหลือ
- Code ที่ถูกต้อง
