## Prompt

ใช้ workflow นี้เมื่อต้องการใช้ภาษาไทยในการสื่อสารและทำงาน

## Execute

1. Set Language Context

- ตั้งค่าใช้ภาษาไทยเป็นหลัก
- อธิบาย concepts เป็นภาษาไทย
- ใช้คำศัพท์เทคนิคที่เหมาะสม
- รักษาความชัดเจน

2. Communicate in Thai

- ตอบเป็นภาษาไทย
- อธิบาย technical terms เป็นภาษาไทย
- ใช้ตัวอย่างที่เข้าใจง่าย
- รักษา professionalism

3. Documentation

- เขียน documentation เป็นภาษาไทย
- ใช้ code examples ที่เหมาะสม
- อธิบายด้วยภาษาที่เข้าใจง่าย
- รักษา consistency

## Rules

1. Clear Communication

- ใช้ภาษาที่ชัดเจน
- หลีกเลี่ยง slang
- อธิบาย technical terms
- รักษา professionalism

2. Technical Accuracy

- ใช้คำศัพท์เทคนิคที่ถูกต้อง
- ไม่แปลคำที่ควรใช้ภาษาอังกฤษ
- รักษาความถูกต้องทางเทคนิค
- Code ยังใช้ภาษาอังกฤษ

3. Accessibility

- เข้าใจง่ายสำหรับคนไทย
- อธิบาย concepts ละเอียด
- ใช้ตัวอย่างที่ relatable
- รองรับทุกระดับ

## Expected Outcome

- การสื่อสารเป็นภาษาไทยที่ชัดเจน
- ความเข้าใจที่ดีขึ้น
- Documentation ที่เข้าถึงได้
- ผลงานที่มีคุณภาพ
