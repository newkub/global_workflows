## Goal

สรุปข้อมูลกระชับให้เข้าใจง่ายและอ่านได้รวดเร็ว

## Scope

ใช้สำหรับการสรุป:
- Quick overviews
- Executive summaries
- Status updates
- Progress reports

## Execute

### 1. Extract Key Points

1. ระบุ key metrics และ statistics
2. ระบุ critical issues และ blockers
3. ระบุ achievements และ milestones
4. ระบุ next steps และ actions

### 2. Format Summary

1. ใช้ bullet points สำหรับ quick scanning
2. ใช้ bold สำหรับ keywords สำคัญ
3. ใช้ sections สำหรับ grouping
4. ใช้ status indicators (✅, ❌, ⚠️)

### 3. Add Context

1. เพิ่ม timestamp สำหรับ reference
2. เพิ่ม scope สำหรับ context
3. เพิ่ม links สำหรับ details
4. เพิ่ม owner หรือ responsible party

### 4. Prioritize Information

1. แสดง critical items ก่อน
2. ใช้ hierarchy สำหรับ importance
3. ใช้ concise language
4. ใช้ action-oriented phrasing

## Rules

### Brevity

ความกระชับ

- ใช้ short sentences และ bullet points
- หลีกเลี่ยง unnecessary details
- ใช้ active voice
- ใช้ clear language

### Structure

โครงสร้างที่ชัดเจน

- ใช้ logical grouping
- ใช้ consistent formatting
- ใช้ clear headings
- ใช้ visual hierarchy

### Actionability

การกระตุ้นให้ทำ

- ใช้ action-oriented language
- ระบุ owners และ deadlines
- ใช้ clear next steps
- ใช้ specific metrics

## Expected Outcome

- Summary ที่กระชับและอ่านง่าย
- Key points ที่ถูก highlight
- Context ที่เพียงพอ
- Action items ที่ชัดเจน
