## Goal

Review UX/UI ทั้งหมดเพื่อระบุและแก้ไข issues ตาม priority

## Scope

ครอบคลุม usability, accessibility, responsive design, user flows

## Execute

### 1. Analyze User Flows

1. ตรวจสอบ user flows
2. Review navigation structure
3. Check CTAs placement
4. Validate user journey

### 2. Review Interface Design

1. Check visual hierarchy
2. Review spacing and layout
3. Validate color usage
4. Check typography

### 3. Check Accessibility

1. Review keyboard navigation
2. Check screen reader support
3. Validate color contrast
4. Check ARIA labels

### 4. Review Responsive Design

1. Check mobile experience
2. Review tablet layout
3. Validate desktop experience
4. Check breakpoints

### 5. Prioritize And Report

1. Group issues ตาม impact
2. ใช้ `/report` เพื่อสรุปผล
3. รอ user confirmation

### 6. Execute Fixes

1. Fix critical UX issues
2. Improve accessibility
3. Fix responsive issues
4. Improve user flows

## Rules

### 1. User Flows

ตรวจสอบ user flows:

- User flows ชัดเจน
- Navigation intuitive
- CTAs prominent
- Feedback immediate
- Error messages ชัดเจน

### 2. Interface Design

ตรวจสอบ interface design:

- Visual hierarchy ดี
- Spacing consistent
- Colors ใช้ถูกต้อง
- Typography ดี
- Loading states ชัดเจน

### 3. Accessibility

ตรวจสอบ accessibility:

- Keyboard navigation ใช้ได้
- Screen reader support
- Color contrast ดี
- ARIA labels ครบถ้วน
- Focus states ชัดเจน

### 4. Responsive Design

ตรวจสอบ responsive design:

- Mobile experience ดี
- Tablet layout ดี
- Desktop experience ดี
- Breakpoints ถูกต้อง
- Touch targets ดี

### 5. Usability

ตรวจสอบ usability:

- Interface intuitive
- Learning curve ต่ำ
- Error prevention
- Recovery ง่าย
- Consistent patterns

## Expected Outcome

- UX ดีขึ้น
- Accessibility ผ่าน
- Responsive design ดี
- User flows ดีขึ้น
- Usability สูง

## Common Mistakes

ข้อผิดพลาดที่พบบ่อย:

- ไม่ accessible
- Mobile experience แย่
- Navigation สับสน
- ไม่มี feedback
- Error messages ไม่ชัดเจน

## Anti-Patterns

รูปแบบที่ไม่ควรทำ:

- ❌ ไม่ accessible
- ❌ Mobile experience แย่
- ❌ Navigation สับสน
- ❌ ไม่มี feedback
- ❌ Error messages ไม่ชัดเจน
