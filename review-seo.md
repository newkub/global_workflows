## Goal

Review SEO ทั้งหมดเพื่อระบุและแก้ไข issues ตาม priority

## Scope

ครอบคลุม meta tags, content structure, technical SEO, performance

## Execute

### 1. Analyze Meta Tags

1. ตรวจสอบ title tags
2. Review meta descriptions
3. Check Open Graph tags
4. Validate structured data

### 2. Review Content Structure

1. Check heading hierarchy
2. Review content quality
3. Analyze keyword usage
4. Check internal linking

### 3. Check Technical SEO

1. Review sitemap.xml
2. Check robots.txt
3. Validate canonical URLs
4. Check URL structure

### 4. Review Performance

1. Check page speed
2. Review mobile experience
3. Analyze Core Web Vitals
4. Check HTTPS

### 5. Prioritize And Report

1. Group issues ตาม impact
2. ใช้ `/report` เพื่อสรุปผล
3. รอ user confirmation

### 6. Execute Fixes

1. Fix critical SEO issues
2. Improve meta tags
3. Fix technical SEO
4. Optimize content structure

## Rules

### 1. Meta Tags

ตรวจสอบ meta tags:

- มี title tags ทุกหน้า
- มี meta descriptions
- มี Open Graph tags
- มี Twitter cards
- มี canonical URLs

### 2. Content Structure

ตรวจสอบ content structure:

- ใช้ heading hierarchy ถูกต้อง
- มี content คุณภาพสูง
- ใช้ keywords อย่างเหมาะสม
- มี internal linking
- มี multimedia

### 3. Technical SEO

ตรวจสอบ technical SEO:

- มี sitemap.xml
- มี robots.txt
- ใช้ HTTPS
- มี structured data
- URL structure ดี

### 4. Performance

ตรวจสอบ performance:

- Page speed ดี
- Mobile-friendly
- Core Web Vitals ผ่าน
- ไม่มี broken links
- ใช้ CDN

### 5. Accessibility

ตรวจสอบ accessibility:

- มี alt text
- ใช้ semantic HTML
- มี skip links
- มี ARIA labels
- Color contrast ดี

## Expected Outcome

- SEO score สูง
- Meta tags ครบถ้วน
- Technical SEO ถูกต้อง
- Performance ดี
- Ranking ดีขึ้น

## Common Mistakes

ข้อผิดพลาดที่พบบ่อย:

- ไม่มี meta tags
- Heading hierarchy ผิด
- ไม่มี sitemap
- Page speed ช้า
- ไม่ mobile-friendly

## Anti-Patterns

รูปแบบที่ไม่ควรทำ:

- ❌ ไม่มี meta tags
- ❌ Duplicate content
- ❌ ไม่มี sitemap
- ❌ Page speed ช้า
- ❌ ไม่ mobile-friendly
