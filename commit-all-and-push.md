## Goal

Commit ทุกไฟล์และ push ไปยัง remote repository

## Execute

### 1. Commit All Changes

สร้าง commit สำหรับทุกไฟล์ที่มีการเปลี่ยนแปลง

1. ทำตาม `/commit-all`

### 2. Push Changes

Push commits ไปยัง remote repository

1. ทำตาม `/git-push`

### 3. Update References

อัปเดท references ทั้งหมดที่เกี่ยวข้อง

1. ทำตาม `@[/edit-relative]`

## Expected Outcome

1. ทุก changes ถูก commit ตาม conventional commits
2. Changes ถูก push ไปยัง remote repository
3. Git history สะอาดและติดตามง่าย
