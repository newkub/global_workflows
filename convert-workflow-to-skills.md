## Goal

แปลง workflow files เป็น skill files โดยรักษาเนื้อหาและปรับโครงสร้างตาม skill format

## Scope

ใช้สำหรับการแปลง workflow files เป็น skill files ใน skills repository

## Execute

1. เลือก workflows ที่ต้องการแปลง
2. ทำตาม `/write-windsurf-skills`

## Rules

- ตรวจสอบว่า `/write-windsurf-skills` มีอยู่จริงก่อนอ้างอิง

## Expected Outcome

- Workflows แปลงเป็น skills สำเร็จ
- Frontmatter ถูกต้องตาม skill format
- Sections restructure ตาม skill format
- Content เหมาะสมกับ skills
- Supporting files ครบถ้วน
