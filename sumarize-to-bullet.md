## Goal

สรุปเนื้อหาให้อยู่ในรูปแบบ bullet points

## Scope

ใช้สำหรับการสรุปเนื้อหาใดๆ ให้อยู่ในรูปแบบ bullet points

## Prompt

ใช้ workflow นี้เมื่อต้องการสรุปเนื้อหาให้อยู่ในรูปแบบ bullet points

## Execute

1. Read and Analyze Content

- อ่านเนื้อหาต้นฉบับ
- ระบุ main points
- หา key takeaways
- วิเคราะห์ structure

2. Extract Key Points

- แยกส่วนที่สำคัญ
- ลบรายละเอียดที่ไม่จำเป็น
- รักษา context
- จัดลำดับตาม priority

3. Format as Bullets

- ใช้ concise language
- หนึ่ง bullet ต่อหนึ่ง point
- ใช้ consistent formatting
- Group related points

4. Review and Refine

- ตรวจสอบความครบถ้วน
- ยืนยันว่าไม่ missing information
- Refine language
- สรุป final version

## Rules

1. Clarity

- หนึ่ง bullet = หนึ่ง idea
- ใช้ภาษากระชับ
- ไม่ซับซ้อนเกินไป
- Easy to scan

2. Completeness

- รวมทุกสาระสำคัญ
- ไม่ distort ความหมาย
- Maintain context
- Include all key points

3. Structure

- จัดกลุ่มที่เหมาะสม
- ใช้ hierarchy ถ้าจำเป็น
- Consistent formatting
- Logical flow

## Expected Outcome

- สรุปในรูปแบบ bullet points
- เนื้อหาครบถ้วนและกระชับ
- Easy to read and understand
- สรุปที่มีประโยชน์
