## Goal

แก้ไขไฟล์เดิมให้ดีขึ้นตาม best practices จาก official website และเอกสารที่เป็นทางการ

## Execute

### 1. Apply Best Practices

1. ทำตาม `/learn-best-practices`
2. ระบุไฟล์ที่ต้องการแก้ไขใน project
3. แก้ไขไฟล์เดิมให้สอดคล้องกับ best practices ที่เรียนรู้มา

## Rules

- ใช้ `/learn-best-practices` เป็น primary workflow
- แก้ไขไฟล์เดิม ไม่สร้างไฟล์ใหม่
- บันทึก sources ทั้งหมด
- เขียนเป็นภาษาไทย

## Expected Outcome

- ไฟล์ที่แก้ไขแล้วสอดคล้องกับ best practices
- Best practices ที่ถูกต้องและเป็นปัจจุบัน
- Documentation ที่บันทึกไว้สำหรับ future reference
