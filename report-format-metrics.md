## Goal

แสดง metrics และ statistics ให้ชัดเจนและอ่านง่าย

## Scope

ใช้สำหรับการแสดง:
- Performance metrics
- Test coverage
- Usage statistics
- Build statistics

## Execute

### 1. Collect Metrics

1. รวบรวม metrics จาก sources ต่างๆ
2. ตรวจสอบ data types และ units
3. จัดกลุ่ม metrics ตาม categories
4. จัดเรียง metrics ตามความสำคัญ

### 2. Format Metrics

1. ใช้ tables สำหรับ structured metrics
2. ใช้ charts สำหรับ visual metrics
3. ใช้ progress bars สำหรับ percentages
4. ใช้ sparklines สำหรับ trends

### 3. Add Context

1. เพิ่ม baseline values สำหรับ comparison
2. เพิ่ม thresholds สำหรับ alerts
3. เพิ่ม trends สำหรับ historical data
4. เพิ่ม annotations สำหรับ insights

### 4. Highlight Issues

1. ใช้ colors สำหรับ status (green, yellow, red)
2. ใช้ icons สำหรับ alerts
3. ใช้ bold สำหรับ critical metrics
4. ใช้ summaries สำหรับ quick overview

## Rules

### Data Types

ประเภทข้อมูลที่ต้องจัดรูปแบบ

- numerical values พร้อม units
- percentages พร้อม progress indicators
- timestamps พร้อม relative time
- categorical data พร้อม labels

### Visualization

การแสดงผลแบบ visual

- ใช้ tables สำหรับ structured data
- ใช้ progress bars สำหรับ percentages
- ใช้ sparklines สำหรับ trends
- ใช้ colors สำหรับ status indicators

### Context

บริบทที่ควรมี

- baseline values สำหรับ comparison
- thresholds สำหรับ alerts
- trends สำหรับ historical context
- annotations สำหรับ insights

## Expected Outcome

- Metrics ที่ชัดเจนและอ่านง่าย
- Visualizations ที่เหมาะสมกับ data types
- Context ที่เป็นประโยชน์
- Issues ที่ถูก highlight
